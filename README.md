# Proximity Microservice

## Overview:

A microservice used to discover nearby places such as restaurants, hotels, theaters, museums,etc. Users can add their own business and add review to other buinesses.

## Technologies:

- Geohash
- RabbitMQ
- Redis
- PostgreSQL
